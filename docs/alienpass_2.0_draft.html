AlienPass v2.0 Draft

Mnemonic command string generator

This draft implements the v2.0 stateless generator exactly as defined in the specification: one login, one command string, PBKDF2-HMAC-SHA256 at 600,000 iterations, and deterministic formatting from fixed alphabets.

Input model modifier:secret with defaults for alphabet, length, and version.
Key derivation PBKDF2 with salt Login|Version and 64 output bytes.
Formatting Guaranteed character classes for default and abc modes, digits only for pin.

Generator

Use the live parser preview before generating the password.

Examples: AmazonTower, v2:GmailTower, pin4:MyBankTower.

Alphabet: default  |  Length: 14  |  Version: v1

The salt is built as Login|Version.

Specification summary

The draft below matches the current v2.0 mnemonic specification.

Grammar

^(?:(abc|pin)?(\d{1,2})?(v\d+)?\:)?(.*)$

Defaults

  • Alphabet: default
  • Length: 14
  • Version: v1

Alphabets

  • default: upper, lower, digit, symbol, then mixed
  • abc: upper, lower, digit, then mixed letters and digits
  • pin: digits only

Core

  • PRF: HMAC-SHA256
  • Iterations: 600000
  • Output: 64 bytes
